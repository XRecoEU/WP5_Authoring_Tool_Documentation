ModelViewer Template Scene

Overview

The ModelViewer Template Scene is designed to help users to get a closer look at certain 3d models in a scene. This scene can be built as a desktop and mobile application.

Contents

Scene Component or Prefabs inside the Scene

The template uses an Orbit Camera for intuitive navigation around the 3D model. The camera supports mouse and touch controls for zooming, rotating, and panning.

Example 3D Model

An example 3D model is included in the scene. This model can be replaced with your own 3D assets. Simply import your model into the project and update the scene accordingly. Use the ContentPlacementHelper-Plugin to place and manipulate the model as you like.

UI Text

The scene includes an UI Text Prefab which is designed to help users with displaying text on the screen. It concentrates all important settings for text in one place. Size and Position can be adjusted via the Transform Component, visit the the UI Text Documentation for more informations.

Setup Instructions

  1. Open the ModelViewer Template Scene in Unity.
  2. Replace the example 3D model with your own by dragging your model into the scene and aligning it as needed.
  3. Customize the Ui Text by modifying the prefab instances.
  4. Adjust the Orbit Camera settings if necessary to match your desired interaction experience.

Customizing the Scene

  • Replace Example Model: Delete the placeholder 3D model and add your own assets. Ensure the model is scaled and positioned appropriately.
  • Camera Adjustments: Use the Inspector to fine-tune the Orbit Camera's zoom limits, rotation speed, and other parameters.
  • Scene lightning: Use the light intensity scrollbar to select the perfect lightning for your scene.

Best Practices

  • Keep your 3D model optimized for performance, especially for mobile and web platforms.
  • Use annotations to highlight key features or provide additional context for your model.
  • Test the scene on all target platforms to ensure consistent performance and usability.
