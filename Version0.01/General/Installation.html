Installation of the XReco Authoring Tool

We assume you already have a recent version of Unity 6
(we would recommend version 6000.1.4 but a newer version of unity 6.1 will also work just fine)
installed on your machine, if not please do so before your installen the XReco Authoring Tool.
You will need admin rights to install unity, instructions can be found here.

Where to get the data

You can download some sample assets (3d, audio, image) from here.
But you can also use your own 3d assets.

You can then download and extract the XReco Authoring Tool from here.

How to Get Started

Install the XReco Authoring Tool by importing the package into your Unity project.
Explore the available Template Scenes and Prefabs in the XReco folder.
Check out the documentation pages for each component to understand their setup and usage.
