Welcome to the XReco Authoring Tool

Thank you for choosing the XReco Authoring Tool! This tool is designed to enhance your Unity development experience by providing pre-built Template Scenes, Prefabs, and Custom Scripts that simplify and accelerate the creation of interactive 3D experiences.

What is the XReco Authoring Tool?

The XReco Authoring Tool extends the Unity Editor with a set of powerful tools and resources that streamline common tasks in interactive development. Whether you're creating educational content, product visualizations, or immersive experiences, this tool gives you a head start with reusable components and intuitive workflows.

Key Features

  • Template Scenes: Ready-to-use scene templates for common use cases such as infographics, virtual production and model viewer for 2D/AR.
  • Prefabs: Pre-configured, reusable objects such as annotations, navigation systems, and interactive elements.
  • Custom Scripts: Scripts that handle essential functionalities like camera controls, video playback/recording and 3d-Asset-Placement.
  • Ease of Use: Intuitive integration with the Unity Editor to make advanced features accessible to everyone.

How to Get Started

  1. Install the XReco Authoring Tool by importing the package into your Unity project.
  2. Explore the available Template Scenes and Prefabs in the XReco folder.
  3. Check out the documentation pages for each component to understand their setup and usage.

For more detailed instructions, visit the full documentation.
