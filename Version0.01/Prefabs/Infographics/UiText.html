UI Text Prefab

Overview

The UI Text Prefab is designed to help users with displaying text on the screen. It concentrates all important settings for text in one place. Size and Position can be adjusted via the Transform Component.

Inspector Properties

  • Text: For Texteditiing.
  • Font Size: To change the text size.
  • Font Style: To change the text style.
  • Text Color: To change the text color.
  • Alignment: To change the text alignment.
  • Postion: To change the text postion.

How to Use

  1. Add the UI Text Prefab component in your scene (An Canvas should be also added to your scene).
  2. Use the TextHelper to adjust your text.
  3. For more advanced control, go directly to the TextMeshPro - Text component
