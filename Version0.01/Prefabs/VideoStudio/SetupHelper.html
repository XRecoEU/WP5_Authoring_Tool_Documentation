Setup Helper

Overview

The Setup Helper is designed to help users setting up a scene by de/activating multiple objects at a time.

Inspector Properties

  • Objects To Activate: list of Gameobjects that will be activated
  • Objects To De Activate: list of Gameobjects that will be deactivated
  • Camera
  • Button "Activate": activates all the objects in the activate list and deactivates all the objects in the deactivate list
  • Button "Deactivate": activates all the objects in the deactivate list deactivates all the objects in the activate list
  • Button "Toggle": does one of the above, if Activate was last then Deactivare or the other way round.

How to Use

  1. Add the Setup Helper component to an empty GameObject in your scene.
  2. Add one set of GameObjects from your scene to the Objects To Activatelist.
  3. Add another set of GameObjects from your scene to the Objects To De Activatelist.
  4. switch GameObjects in your scene as needed.
