Content Placement Helper Prefab

Overview

The Content Placement Helper Prefab is designed to help users to place 3d content in the scene. Just drop an asset from the "Project" view into the "Content to Place" field in the "Content Placement Helper" component. It will then automatically scaled and positioned so that it fills the spcified Area.

How to Use

  1. Add the Content Placement Helper Prefab component to your scene.
  2. Specify the 3D asset you want to place in the Content To Place field.
  3. Adjust the bounding box dimensions by scaling the GameObject.
  4. Select the desired Model Anchor Position (Top, Middle, or Bottom).
  5. Call the SpawnModel method (e.g., through a script or via the Unity Editor) to place and scale the asset within the bounding box.
  6. Use the DeSpawnModel method to remove the asset if needed.
