Recording Helper Prefab

Overview

The Recording Helper Prefab is designed to help users to record video data.

Inspector Properties

  • Video File Name: The name for the resulting file
  • Make Name Unique: If checked and the name is allready taken there will be a number appended to the name.
  • Resolution: The X and Y resolution for the video.
  • Frame Rate: The frame rate for the video.
  • Start Recording On Enabled: If checked the recording will start themoment the Gameobject is enabled.
  • Record Audio: If checked the Audio data will be included in the video.

How to Use

  1. Add the Recording Helper Prefab GameObject to your scene.
