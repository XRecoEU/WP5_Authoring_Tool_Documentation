Menu

Overview

The Menu provides direct access to important functions, documentation and parter Websites

Available Menu points with short description

  • Actions - funktions to help place Gameobjects
    • Atach to Cursor
    • Atach to Cursor and Rotate to Cam
    • Rotate to Cam
  • Backup
    • Backup Project as zip
  • Build - Build funktions for diffferent target platforms
    • Build and Run Scene For Web
    • Build Scene For Android
    • Build Scene For Desctop
    • Build Scene For iOS
    • Build Scene For MacOS
    • Open Build Folder
  • Export
    • Current Scene as unitypackage
    • Whole Project as unitypackage
    • Whole Project with ProjectSettings as unitypackage
  • Help
    • Open Authoring Tool Introduction Video
    • Open Documentation
    • Open Get Started - Usertest
    • Open Tutorials
    • Open Unity Documentation
    • Open Welcome Screen
  • Partner Sites - this area provides direct links to Partner Websites
    • Open FVV Page
    • Open Holoportation Page
    • Open XRCapsules Page
    • Open XREco Dashboard
    • Open XREco Homepage
  • Collapse non-XREco Components - toggle to collapse or expand Components in the inspector Tab
  • Import Asset - opens the Import Asset window whit options to drag'n'drop or use a drectory explorer to import Asset data
  • Performance Indicator - opens the Platform Indicator Window to see workload stats
  • Prefabs Browser - opens the Prefabs Browser popup window
