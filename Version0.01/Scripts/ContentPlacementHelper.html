Content Placement Helper

Overview

The Content Placement Helper is a Unity component designed to simplify the placement of 3D assets within predefined areas. This tool ensures that the assets are appropriately scaled and centered within the designated area, making it easier to manage and display content in your scene.

  • Allows for the placement of a 3D asset within a predefined box.
  • Automatically scales the asset to fit the bounding box.
  • Supports aligning the asset to the top, middle, or bottom of the box.
  • Dynamic bounding box calculation for accurate scaling and placement.
  • Visual representation of the bounding box and asset placement using Gizmos in the Unity Editor.

Inspector Properties

  • Content To Place: The 3D asset to be placed in the scene.
  • Model Anchor Position: Determines the vertical alignment of the asset within the bounding box (Top, Middle, Bottom).
  • Model Instance (Read-Only): The instantiated 3D asset in the scene.

How to Use

  1. Add the Content Placement Helper component to an empty GameObject in your scene.
  2. Specify the 3D asset you want to place in the Content To Place field.
  3. Adjust the bounding box dimensions by scaling the GameObject.
  4. Select the desired Model Anchor Position (Top, Middle, or Bottom).
  5. Call the SpawnModel method (e.g., through a script or via the Unity Editor) to place and scale the asset within the bounding box.
  6. Use the DeSpawnModel method to remove the asset if needed.

Editor Gizmos

The component uses Gizmos to visualize the bounding box and the placement of the asset in the Unity Editor:

  • The bounding box is shown as a purple translucent box.
  • The placed asset's bounding box is shown as a green translucent box.
