XReco Comment

Overview

The XReco Comment is a Unity component designed to add a custom color to the hierarchy background of the referenced component.

  • Automatically mappes the documentation of the referenced component to the open Documentation Button.

Inspector Properties

  • Button: --> Open Documentation this opens the documentation of the referenced component
  • Button: Edit Opens the edit window for the referenced component. Here can the reference or some of its attributes be edited.

How to Use

  1. Add the XReco Comment component to an empty GameObject in your scene.
  2. Add another XReco component to the GameObject
