Green Screen Video Helper

The Green Screen Video Helper component allows you to manage and display green screen videos within your Unity scenes. It provides tools for loading, preparing, and displaying videos on a material with real-time updates and customization options.

Inspector Properties

Core Properties

Green Screen Video: The video clip to be displayed on the green screen.
Video Player: The VideoPlayer component used to play the video.
Green Screen Material: The material used to render the green screen video.
Green Screen Renderer: The MeshRenderer that displays the green screen video.

Internal Settings

Green Screen Texture: A dynamically created render texture that matches the dimensions of the video.

How It Works

The Green Screen Video Helper component handles the following tasks:

  • Video Playback: Loads and plays a green screen video on a material.
  • Texture Management: Creates a render texture to match the video's dimensions and applies it to the material.
  • Frame Preview: Displays a specific frame of the video based on a normalized time value.
  • Aspect Ratio Adjustment: Automatically scales the renderer to match the aspect ratio of the video.

Setup Instructions

  1. Attach the GreenScreenVideoHelper script to a GameObject in your scene.
  2. Assign a VideoClip to the Green Screen Video property in the Inspector.
  3. Ensure the GameObject contains the following components:
    • A VideoPlayer component.
    • A MeshRenderer with a material to display the video.
  4. Configure the green screen material as needed. The video will be applied as the material's main texture.

Using the Component

  • Show Frame: Call the ShowFrame(float normalizedTime) method to display a specific frame of the video. The normalizedTime parameter is a value between 0 (start of the video) and 1 (end of the video).
  • Prepare Video Player: The component prepares the video player on startup and can adjust settings dynamically via the PrepareVideoPlayer(bool showFirstFrame) method.

Tips

  • Ensure the Green Screen Video property is assigned a valid video clip.
  • Use the Refresh button in the custom editor to update the green screen setup if you make changes.
  • For optimal performance, test your setup in the Unity Editor and on your target platform.
